# Changelog

All notable changes to this project will be documented in this file.

## [2.7.0] - 2025-01-XX

### Fixed
- Fixed "Row not found on page" error when applying classifications
- Books are now tracked with their page numbers during scanning
- Apply now only processes books on the current page
- Added skip notification for books on other pages

### Changed
- Improved multi-page scanning reliability
- Better error messages and user feedback

## [2.6.0] - 2025-01-XX

### Added
- Multi-page scanning with automatic navigation
- Wait for page content to load before scanning
- Persistent scan state across page navigations

### Fixed
- Books on pages other than the first page are now correctly detected

## [2.5.1] - 2025-01-XX

### Initial Release
- AI-powered book classification
- Multi-page library support
- Batch processing
- Auto-apply to collections
- Persistent configuration
- Clean draggable UI
